# iMagdy Homebrew Tap

Homebrew tap for my command-line tools.

## Install skm

After an `skm` release publishes its formula:

```bash
brew install imagdy/tap/skm
```

Or tap the repository first:

```bash
brew tap imagdy/tap
brew install skm
```

Formulae live under `Formula/` and are updated by each project's release workflow.
